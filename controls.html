控制语句

条件控制语句

根据布尔变量的值作为判断形成了if语句的判断原型，可以从最简单的语句开始。

temp = 37.5
if temp >= 38:
    print("你已经发烧了!")
else:
    print("你的体温正常")
    print("注意保持!")
你的体温正常
注意保持!

从上面的例子中，通过简单的if ... else ...语句判断temp是否高于38，打印相应的语句。

多重条件

第一个例子中只有一个判断条件：即输入变量是否大于38。如果有多个判断条件，比如接着上面的例子通过温度再细分，可以用到elif去添加更多的判断条件。

temp = 38.5
if temp > 40:
    print("哦！小心，这是高烧!")
elif temp >= 38:
    print("嗯！这是低烧。")
elif temp >= 37:
    print("测试")
else:
    print("你的体温正常")
嗯！这是低烧。

课后练习: 设计一段程序，完成下面任务：根据用户输入的成绩，打印不同的成绩结果，其中大于90，打印A, 80-90; 打印B, 70-79, 打印C; 60-69,打印D;其余的打印F。 包括了input，if...elif的使用,还有输入格式必须是数字。

Match语句

Python 3.10以后，引入Match语句，可以根据变量值来判断进入不同的分支。

day = "Sunday"
match day.lower():
        case "monday" | "tuesday" | "wednesday" | "thursday" | "friday":
            print("Weekday")
        case "saturday" | "sunday":
            print("Weekend")
        case _:
            print("Invalid day")
Weekend

课后练习: 设计一个程序使用match语句， 根据不同的HTTP状态码来判断显示不同的提示语句，当状态码是404，显示”not found”; 当状态码是400，显示”bad request”;当状态码是500, 显示”Internal server error”;其他显示”Something wrong with the service”

逻辑条件判断

通过判断条件的组合，if语句可以完成多个变量的同时判断。

age = 18
sex = "F"
if age >= 18 and sex == "F":
    print("成年女性")
成年女性

三元运算符

简单的逻辑判断可以通过简单的三元运算符来实现。

x = 3242389
"偶数" if x % 2 == 0 else "奇数"
'奇数'
x
3242389

多重循环

for i in range(10):
    if i == 5:
        break
    print(i)

continue语句

如果在循环中，遇到满足某些条件需要跳出当前循环，采用continue语句可以当前目标。同样的，continue也是用于跳出最内层的循环。

for i in range(10):
    if i == 5:
        continue
    print(i)

将前面例子中的break替换为continue，我们可以看到当i值为5时，跳过了打印的操作直接进入下一个循环。

与else结合的循环语句

在Python中，循环语句有一个特殊的else语句，当循环正常执行完以后会执行else语句。

for i in range(5):
    print(i)
else:
    print("循环正常结束。")

while循环

类似于for循环，while循环是另一种常见的循环控制语句。

import time

seconds = 5
while seconds > 0:
    print(seconds)
    time.sleep(1)
    seconds -= 1
print("时间到!")
5
4
3
2
1
时间到!

与for循环一样，while循环可以结合break和continue一起使用。

num = 0
while True:
    num += 1
    if num % 3 == 0:
        continue
    if num > 50:
        break
    print(num, end=" ")

print("\n结束操作")
